# Slash command

```js
TextBasedChannel.sendSlash(
    user: BotId (Snowflake) | User (User.bot === true),
    commandName: 'command_name [sub_group] [sub]',
    ...args: (string|number|boolean|FileLike|undefined)[],
): Promise<Message<true> | Modal>
```
